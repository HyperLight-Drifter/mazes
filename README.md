# Mazes — Unofficial Foundry VTT System

An unofficial system for running Mazes in Foundry VTT. This project is not affiliated with, endorsed by, or connected to the creators, 9th Level Games. Find out more about Mazes RPG at: https://9thlevel.com/pages/mazes-fantasy-roleplaying

AI usage: Claude AI was involved in creation of the system. 

## Features

### Actors
- **Character sheet** with portrait, name, and concept
- Resource trackers for Hearts, Stars, Gears, Lore, and Wealth and Lifestyle
- Role, Aspect, and Class item slots with drag-and-drop support
- Edge items with expandable descriptions
- Action rolls for Books, Boots, Blades, and Bones with Advantage / Normal / Disadvantage dialogue
- Effect rolls with exploding dice and Advantage / Normal / Disadvantage
- Chaos rolls with Advantage / Normal / Disadvantage
- Healing and Death's Door rolls with full resolution text
- **Conditions** tab with togglable active state by either clicking, or inputting custom text
- **Campaign** tab with company name, drive, drive die and drag-and-drop zones for hexcrawl and downtime actions. Rollbuttons for said actions.
- **Milestones** tab with checkable, strike-through milestone entries
- **Notes** tab

- **Hazard sheet** 
- name, Danger and Hearts trackers
- Edge items in two-column layout
- Description 
- Conditions tab matching the character sheet

### Items
- **Role** — die size (d4/d6/d8/d10), description, derived Hearts and Stars display
- **Aspect** — description
- **Class** — description
- **Edge** — description
- **Campaign Action** - items for hexcrawl and downtime actions with roll automation

### Game Sheet
- Persistent floating widget visible to all connected users
- Tracks Darkness with category label (Bright / Torchlit / Bleak) and commentary
- Tracks Treasure with player count, category label (Skint / Flush / Encumbered) and commentary
- Tracks Supply
- GM-editable, player-visible
- Per-user repositionable, minimizable

### Combat tracker
- upon clicking Start combat select which side of conflict goes first - Players or Hazards

## Compatibility
- Foundry VTT v14
